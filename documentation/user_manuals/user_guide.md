# User Guide

## What This System Does

Monitors:
- Heart Rate (BPM)
- Body Temperature (°C)
- Blood Pressure (Systolic/Diastolic)

## How to Use

1. Power on device
2. Open mobile app
3. View vitals in real-time
4. Historical data available in “History” tab

## Safety Notes

- Device is non-invasive
- For medical-grade accuracy, consult a physician
